# Prospello V2

Recovered, deployable snapshot of the public Prospello Technologies website from the live Vercel deployment.

## Local preview

```bash
npx serve .
```

## Deployment

This repository is a static site and can be deployed directly to Vercel with no build command.

## Recovery scope

The public HTML, stylesheets, JavaScript, images, and linked pages were recovered. Private server-side source code, environment variables, unpublished routes, and deployment history are not available from a public deployment.
